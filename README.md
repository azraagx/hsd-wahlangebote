  # HSD Wahlangebote - Verwaltungstool für Lehrende

Diese Progressive Web App (PWA) wurde für das Modul "Interaktive Systeme" entwickelt. 
Sie dient Lehrenden zur einfachen Erstellung und Verwaltung von Wahlangeboten.

## Für Gruppenmitglieder: So startest du die App lokal

1. Klone dieses Repository auf deinen PC.
2. Öffne den Ordner im Terminal (z. B. in VS Code).
3. Installiere alle nötigen Abhängigkeiten mit:
   `npm install`
4. Starte den lokalen Entwicklungs-Server mit:
   `npm run dev`
